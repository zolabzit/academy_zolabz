Curriculum

Beginner Course (3 months)

This course is designed for beginners who want to learn the fundamentals of web development. It covers the basics of HTML, CSS, and JavaScript, and will help you build your first website.

Module 1: Introduction to Web Development
  • History of the web
  • Web technologies and standards
  • Introduction to HTML, CSS, and JavaScript
Module 2: HTML Fundamentals
  • HTML structure and syntax
  • Creating and organizing web pages
  • Using HTML elements and attributes
Module 3: CSS Fundamentals
  • CSS syntax and selectors
  • Styling HTML elements with CSS properties
  • Creating responsive layouts with CSS
Module 4: JavaScript Fundamentals
  • Introduction to JavaScript programming
  • JavaScript syntax and data types
  • Variables, operators, and control flow statements

Project: Build a simple website using HTML, CSS, and JavaScript

Intermediate Course (4 months)

This course is designed for intermediate learners who want to take their web development skills to the next level. It covers advanced HTML, CSS, and JavaScript concepts, along with object-oriented programming and DOM manipulation.

Module 1: Advanced HTML
  • Semantic HTML and accessibility
  • HTML forms and input elements
  • Embedding media and data
Module 2: Advanced CSS
  • CSS positioning and layout techniques
  • CSS preprocessors and frameworks
  • Responsive web design principles
Module 3: Object-Oriented JavaScript
  • Classes, objects, and inheritance in JavaScript
  • Encapsulation and data abstraction
  • Object-oriented programming principles
Module 4: DOM Manipulation
  • The Document Object Model (DOM)
  • Manipulating the DOM with JavaScript
  • Event handling and user interactions

Project: Develop a dynamic web application using HTML, CSS, and JavaScript

Advanced Course (6 months)

This course is designed for advanced learners who want to master web development and build complex web applications. It covers JavaScript frameworks, server-side development, cloud computing, and full-stack web development.

Module 1: JavaScript Frameworks
  • Introduction to popular JavaScript frameworks (React, Angular, Vue.js)
  • Framework architecture and components
  • Building single-page applications (SPAs)
Module 2: Server-Side Development
  • Introduction to server-side technologies (Node.js, PHP)
  • Building web APIs and handling HTTP requests
  • Integrating databases and data storage
Module 3: Cloud Computing
  • Cloud infrastructure and platforms (AWS, Azure, GCP)
  • Deploying web applications to the cloud
  • Scalability and performance optimization
Module 4: Full-Stack Web Development
  • Integrating front-end and back-end technologies
  • Building complete web applications
  • Deployment and maintenance of web projects

Project: Create a full-stack web application using JavaScript frameworks, server-side technologies, and cloud infrastructure
